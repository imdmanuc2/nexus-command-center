# Infrastructure Page

Purpose:
Show how the mining operation is connected.

Should display:
- Internet / router / switch path
- Servers hosting pools
- MiningCore services
- Blockchain nodes
- Pools installed on each server
- ASIC miners assigned to pools
- Device health using green/yellow/red
- Clickable cards with drawer details

Infrastructure is for topology and IT visibility.
Mining performance graphs belong on the Mining page.
Miner inventory/actions belong on ASIC Fleet.
